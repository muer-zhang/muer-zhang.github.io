What does it mean to display inline vs inline blocks?

Feb 12, 2022

Sprint TWO

Inline and inline blocks are different display types of CSS.

When display: inline is generated, it means that elements inside the box will be displayed only the content’s space without any line breaks. Height and weight don’t apply. However, horizontal paddings and margins apply.

When display: inline-box is generated, it means that elements inside the box will looks like inline (no line breaks) and behaves like block (changeable height, width, margins and paddings).
