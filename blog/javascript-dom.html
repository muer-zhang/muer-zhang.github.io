An analogy to describe the differences between HTML and CSS

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML describes the structure of Web pages. It provides display information of various tags to the browser.  While CSS is used to enhance the presentation of Web pages, including colors, layout, and fonts.

For example, we need to build a house. HTML, the builder, comes in and takes care of the basic structure of the house first. While CSS, the designer, takes part in designing. For example, CSS decides where to locate the main door and its color. HTML and CSS both work together to have the house built.

WHAT DOM IS AND & AN EXAMPLE OF HOW YOU MIGHT INTERACT WITH IT:

DOM is short for the Document Object Model. It provide a link to use Java Script to modify a website which ties tit ogether with the use of HTML and CSS.

Let’s use the example of the design of my blog. I use HTML to do the basic structure of the website, including the headings, the content, and pictures etc. Then I use CSS to style the appearance. For example, I changed the background colour of the blog. I’ve added a few different kinds of google fonts. I can position some contents to line up in a row or in a column. And now I am learning to use Java Script which deals with interactive elements, e.g. read to a mouse click, check the checkboxes. At the bottom of my blog, I am thinking of making a like button. I can use HTML and CSS to create the button, style and position it. Then I will use the DOM to access the documents and this element so that I could create the interactivity of the button. The number of times that it’s been clicked will then show up below the like button.

Explain the difference between accessing data from arrays and objects.

To access data from arrays, we use the numerical index.
To access data from objects, we use a dot operator followed by key name. Or we can loop through the object if the key is unknown.

Explain what functions are and why they are useful.

In JavaScript we define a block of code using a specific function name, declare it as a function. The function has a set of instructions to perform a task. The declared procedure will be executed under certain condition. Functions are useful because they can be used repeatedly. They are great to break down different scenarios and to organise the code. Functions enhance the efficiency and readability of a program.
